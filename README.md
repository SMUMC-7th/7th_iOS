# 7th_iOS

7th SMUMC 스터디 iOS

## Commit Style

- Feat : 새로운 기능 추가
- Fix : 버그 수정
- Design : UI(CSS) 수정
- Typing Error : 오타 수정
- Mod : 폴더 구조 이동 및 파일 이름 수정
- Add : 파일 추가 (ex- 이미지 추가)
- Del : 파일 삭제
- Refactor : 코드 리펙토링
- Begin : 프로젝트 세팅
- Merge : 브랜치 합병
- Deploy :  CI/CD 등 배포 관련
- Docs : README나 WIKI 등의 문서 수정

→ 커밋 메시지 [ex- **`Feat : 메인 페이지 (#이슈번호)`**]
